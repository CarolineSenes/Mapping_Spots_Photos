## Tech Stack

**Server:** Node/Express, MongoDB/mongoose
**Client:** React, Mapbox, Material-UI

## Lancement
Clôner le repo. Pour chaque dossier (back et front) :

```bash
# installer les dépendances
$ yarn install

# lancer
$ yarn start
```
